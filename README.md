# glsl-edge-detection 

Gets the intensity of the edge of an image at a given pixel.  Adapted from http://coding-experiments.blogspot.com/2010/06/edge-detection.html
`edge(sampler2D inputTexture, vec2 coord, vec2 imageSize) => [0-1]`
